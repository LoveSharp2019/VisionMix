# VisionMix
视觉混合编程项目,使用cli代理加密部分算法,框架仅提供学习使用,商业用途请自行测试!!!
包含功能:
1. 多品牌相机驱动
2. 模板匹配工具
3. 灰度匹配工具
4. 二维测量
5. 二维码扫描
6. 印刷缺陷检测
7. 其他功能
代理部分项目仅在老版本中存在,此版本需要删除各个模块的代理算子方可使用!
本软件共享仅为学习使用,谢绝索取直接编译运行版本
